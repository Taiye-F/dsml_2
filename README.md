Global Jobs, GDP & Unemployment Analysis (1991–2022):
This project explores global economic and employment trends from 1991 to 2022 using a real-world dataset sourced from Kaggle. It covers GDP, unemployment rates, and employment distribution across agriculture, industry, and services for multiple countries.

Key Features:
- Data Exploration: Identified top 5 countries by average GDP and analyzed their employment patterns.
- Sectoral Shift Analysis: Tracked Nigeria’s transition from agriculture to services.
- Correlation Insights: Found strong positive correlation between GDP and service sector employment.
- Crisis Comparison: Visualized global unemployment during the 2008 financial crisis and COVID-19 pandemic.
- GDP Trajectory: Analyzed Nigeria’s GDP growth using year-over-year changes.
- Global Visualization: Created an animated choropleth map to show GDP evolution worldwide.

- Language: Python
- IDE: VSCode
- Libraries: Pandas, Matplotlib, Seaborn, Plotly Express

Dataset:
- Source: [Kaggle](https://www.kaggle.com/)
- Columns: `Country`, `Year`, `GDP`, `Unemployment Rate`, `Employment in Agriculture`, `Employment in Industry`, `Employment in Services`

Visualizations:
- Bar charts for crisis impact
- Line plots for GDP trends
- Correlation heatmaps
- Animated choropleth map for global GDP.
